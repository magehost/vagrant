MageHost.pro Development Vagrant Box

Usage

  • Install Vagrant
    • Download & Install Vagrant
    • On Mac: sudo mv /opt/vagrant/embedded/bin/curl /opt/vagrant/embedded/bin/curl.disabled
  • Install a provider
    • Parallels Desktop Pro Edition
      • Faster then VirtualBox, especially the shared filesystem
      • About $100 per year
      • Only for Mac
      • Install Vagrant plugin:
        vagrant plugin install vagrant-parallels
    • VirtualBox
      • Free
      • For Mac, Linux, Windows
    • VMware Fusion
      • Commercial, $82 + $79 for the Vagrant plugin
      • For Mac
      • Install Vagrant plugin:
        vagrant plugin install vagrant-vmware-fusion
        Install Licence:
        vagrant plugin license vagrant-vmware-fusion path/to/license.lic
    • VMware Workstation Player [untested]
      • Commercial, $135 + $79 for the Vagrant plugin
      • For Windows and Linux
      • Install Vagrant plugin:
        vagrant plugin install vagrant-vmware-workstation
        Install Licence:
        vagrant plugin license vagrant-vmware-workstation path/to/license.lic
  • Create a working directory and cd into it
  • Install Vagrant hostmanager plugin:
    vagrant plugin install vagrant-hostmanager
  • Get our Vagrantfile:
    curl -OL https://vagrant.magehost.pro/Vagrantfile
  • Start Vagrant box:
    vagrant up
    • On the end of the install you will get some information like the password.
    • If you want to connect to the box via SSH:
      vagrant ssh
    • Running Vagrant on Windows:
      • We recommend to download Git and use the 'Git Bash Here' feature. There you can use vagrant and git commands.
      • We experienced download errors while testing Vagrant version 1.8.x on Windows. On Mac and Linux this version works fine.
      • On Windows we recommend Vagrant 1.7.x. with VirtualBox 5.0.26.
      • Execute before "vagrant up": vagrant plugin install vagrant-winnfsd
